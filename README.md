# 🌐 ChainVerse — Web3 Explorer

**ChainVerse** is an interactive, cyberpunk-themed web application designed to educate and immerse users in the world of Web3, blockchain technology, and cryptocurrencies. It combines visual storytelling, live market data, and a hands-on mining simulator to make complex concepts accessible and engaging.

Built with vanilla HTML, CSS, and JavaScript, ChainVerse runs entirely in the browser with no backend dependencies.

**Live Demo:** [https://create2learn7238.github.io/W3b-and-Rust-14-6-26/](https://create2learn7238.github.io/W3b-and-Rust-14-6-26/)

---

## 📑 Project Pages & Features

The website consists of **4 main interactive pages**:

| Page | Description |
|------|-------------|
| **🏠 Home** | Landing page featuring a blockchain animation, live statistics (blocks mined, BTC supply cap, etc.), and feature cards explaining decentralization, cryptographic security, smart contracts, and Layer 2 solutions like Arbitrum. |
| **📘 Concepts** | Educational hub with side-by-side comparisons: Web2 vs Web3, Ethereum vs Bitcoin, Public Key vs Private Key, and Blockchain vs Traditional Databases. Uses interactive tables and VS grids to clarify core Web3 principles. |
| **📈 Live Prices** | Real-time cryptocurrency price dashboard powered by **CoinGecko API**. Displays current prices, 24h change percentages, market ticker, and price cards for BTC, ETH, SOL, BNB, and MATIC. Includes manual refresh button. |
| **⚙️ Simulator** | Interactive Proof-of-Work mining simulator. Users can modify block data, "mine" blocks by finding a nonce that produces a hash starting with "00", and observe how changing a previous block invalidates subsequent blocks — demonstrating blockchain immutability. |
