How It Works
Step 1 — Data Input
Block index + previous hash + your data + miner address + nonce are concatenated into a string.
Step 2 — SHA-256 Hashing
The string is hashed using the cryptographic SHA-256 algorithm. Output is always 64 hex characters.
Step 3 — Check Target
If the hash starts with N leading zeros (difficulty), we found a valid block! Otherwise nonce++.
Step 4 — Chain It
Valid block is added to the chain. Its hash becomes the previous hash for the next block — immutable!

⬡ Blockchain — 1 Block(s)

⚠ CHAIN INTEGRITY VIOLATED — A block's data was tampered with. The hash of that block no longer matches the prevHash stored in the next block. This demonstrates why blockchain data is effectively immutable — changing any block invalidates all subsequent blocks.
